# 📊 Sales & Revenue Analysis Dashboard

## Overview

This project analyzes supermarket sales data using Python, Pandas, and Matplotlib to generate business insights and visualizations.

## Features

* Sales Performance Analysis
* Revenue Tracking
* Product Line Analysis
* City-wise Revenue Analysis
* Payment Method Distribution
* Monthly Revenue Trends

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Supermarket Sales Dataset

## Key Insights

* Total Revenue Generated
* Best Performing Product Lines
* Revenue Distribution by City
* Customer Payment Preferences

## How to Run

1. Open the notebook in Google Colab.
2. Upload the CSV dataset.
3. Run all cells.
4. View analytics and visualizations.
